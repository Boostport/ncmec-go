# NCMEC Go
Go SDKs for interacting with the National Center for Missing & Exploited Children (NCMEC)'s public APIs.

Available SDKs:
- [Cyber Tipline API Client](cybertipline/)

## Disclaimer
This is not an official NCMEC project nor is it endorsed by NCMEC. This SDK is developed by Boostport and independent
contributors to facilitate interaction with NCMEC's public APIs.

## Supporting the NCMEC
The NCMEC helps find missing children, reduce child sexual exploitation and prevent child exploitation. To find out
more about their mission, please visit the [NCMEC website](https://www.missingkids.org/).

For ways to support the NCMEC, please visit their [Support Us](https://www.missingkids.org/supportus) page.
